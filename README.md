# CryptMyPassword

## Lanzar servidor

```bash
cd server
docker compose up
```

## Lanzar extensión en modo debug

### Firefox
1. Entrar en esta URL
```
about:debugging#/runtime/this-firefox 
```
2. Cargar el manifest.json
3. Ya te funciona todo lo que tiene que funcionar

### Chrome
1. Ve a chrome://extensions en la barra de direcciones.
2. Activa el "Modo desarrollador" en la esquina superior derecha.
3. Haz clic en "Cargar descomprimida" y selecciona la carpeta de tu extensión.
